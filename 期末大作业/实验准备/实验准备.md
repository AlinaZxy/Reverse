# 逆向工程与软件安全实验报告  
## 训练使用WinDbg  
### 调试更改记事本程序显示内容  
#### 实验环境  
* Windows 10  
* WinDbg x64  
* notepad.exe  

#### 实验过程  
1. 手动保存`1.txt`，内容为`ABCD`，用记事本程序打开  
![note](./img/note.png)  
2. 打开WinDbg，`Attach to a Process`到`notepad.exe`  
![Attach to a Process](./img/attach.png)  
3. `!heap -a`查看堆中全部内容  
![查看堆](./img/查看堆.png)  
4. `s -u 1fda8b70000 L000ff000 "ABCD"`在内存中查找`ABCD`字符串  
![查找字符串](./img/查找字符串.png)  
5. 修改对应位置字符串内容，并继续调试  
    ```
    >ezu 000001fd`a8be3770 "hhhh"
    >g
    ```  
    ![结果展示](./img/结果.png)  

6. 更改显示内容成功！
